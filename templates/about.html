Pantry Pal is a search website that gives you recipe ideas based on ingredients you have. Uses Edamam's API for a databank of over 2 million recipes, as well as the search functionality. The search function allows users to apply 55 different filters to allow for special diets, cuisines, and allergies.


Users can make an account to save recipes using the button in the results page.
